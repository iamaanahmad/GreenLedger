- **Primary Green**: #10b981
- **Secondary Green**: #059669
- **Accent Green**: #34d399
- **Dark Text**: #1f2937
- **Light Background**: #f9fafb

### Typography
- **Headings**: Bold, 32-40pt
- **Body**: Regular, 18-24pt
- **Captions**: Light, 14-16pt

### Visual Elements
- Use leaf/plant icons for green theme
- Include charts and graphs for data
- Add screenshots of actual product
- Use consistent spacing and alignment

### Slide Layout
- Keep text minimal (5-7 bullets max)
- Use large, readable fonts
- Include visual elements on every slide
- Maintain consistent branding

## Presentation Tips

### For Judges (Non-Technical)
- Focus on business value, not technical details
- Use concrete examples and metrics
- Emphasize commercial viability
- Show clear ROI for lenders

### Delivery
- Practice timing (aim for 5-7 minutes)
- Speak confidently and clearly
- Make eye contact (if presenting live)
- Be prepared for Q&A

### Key Messages to Emphasize
1. **Problem is real**: Lenders struggle with ESG verification
2. **Solution is practical**: Working prototype, not just concept
3. **Market is huge**: $12T loan market opportunity
4. **Impact is measurable**: 70% time reduction, increased liquidity
5. **Business is viable**: Multiple revenue streams, clear path to scale
